# Blackjack

APCS1_Fall_Final

Text-based game of Blackjack (standard)

Functional:
	Terminal implementation
	Betting
	Scoring

Non-functional:
	GUI implementation
	Multiple turns/rounds
	Separate values for Ace (1/11)

Directions:
	1. javac *.java
	2. java Blackjack
	3. Wager any amount of in-game credit from 1-100 inclusive
	4. Stay (0) or Hit (1)
